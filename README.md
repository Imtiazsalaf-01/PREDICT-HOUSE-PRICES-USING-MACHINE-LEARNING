# -PREDICT-HOUSE-PRICES-USING-MACHINE-LEARNING
As part of Week 1 of the WeIntern program, the "House Price Prediction" project applies machine learning to forecast housing prices. Using Python libraries like NumPy, Pandas, Scikit-learn, Matplotlib, Seaborn, and XGBoost, it delivers an end-to-end, data-driven solution for accurate price estimation.
